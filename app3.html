Aplicación de Reconocimiento de Sonidos

Actualmente estoy desarrollando una aplicación en Android para la detección y reconocimiento de sonidos, que permitirá al usuario entrenar sonidos personalizados y gestionar notificaciones de eventos. La aplicación incluye un backend en Spring Boot, un módulo de IA en Python con redes neuronales para el reconocimiento de sonidos, y utiliza una base de datos MySQL para la persistencia de datos.

  1. Pantalla Principal: Muestra una lista de sonidos reconocidos (como sirenas o bocinas) e incluye opciones para iniciar o detener el reconocimiento de sonidos y acceder a la configuración.
  2. Pantalla de Configuración: Permite al usuario personalizar los sonidos a reconocer y ajustar opciones como la sensibilidad del micrófono y el tipo de notificación visual.
  3. Pantalla de Entrenamiento de Sonidos: El usuario puede grabar nuevos sonidos, enseñárselos a la aplicación y almacenarlos localmente.
  4. Pantalla de Envío de Sonido por WhatsApp: Permite enviar grabaciones de sonido a un contacto de WhatsApp predefinido desde la aplicación.

Nota: La aplicación aún está en desarrollo, por lo que no se encuentran disponibles todas las capturas de pantalla.
